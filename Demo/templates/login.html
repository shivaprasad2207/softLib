About App:

It is book library management tool. Here book library instance is created using the basic details like admin email id and address and phone number.
Once registered, it will it will give him/her a Pin which will be identifier for login to his library. Later he can add books and users who wants to join his/her library.

Admin Roles: Admin can manage user by add/modify/delete. In the book management admin can add book by creating category/subcategory.
Admin can add number of copies available for that book, it will provide identifier for each of book copy. By clicking the copy identifier the user can reserv books.
Admin can do many search operation like per user as how many books reserved and also per book how many copies booked.
Admin can see expired reservation list to alert the user.
User Roles: User can search the book and reserve copy of it, he can see dates of reservation the other users.
